# Approved Text - Word Pool Map

This is a planning reference for the first shared, Farm, Flow, and Dream word pools. The listed powers are metadata only unless marked implemented.

## Design Notes

* Words are not just skins.
* Each word should eventually have base production plus a special rule.
* Upgrades provide general scaling.
* Paths provide playstyle identity.
* Events provide temporary bursts.
* Sentence synergies will later combine word effects.
